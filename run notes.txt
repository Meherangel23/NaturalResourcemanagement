Create new web application Natural Resource Management, with Solar Irrigation & Waterlift System 
  Objective
Design and develop a vibrant, responsive web application that empowers users to explore, customize, and deploy solar-powered irrigation and waterlift systems. The platform should combine rich visuals, interactive tools, and intelligent backend logic to support sustainable agriculture.

 Tech Stack
- Frontend: Angular 17 with Angular Material for UI components and responsiveness
- Backend: FastAPI (Python) for RESTful APIs and chatbot logic
- Database: SQLite or PostgreSQL (optional, for persistent design plans or user data)
- Deployment: Dockerized services, hosted on Render/Vercel/Azure

To run the backend:

Navigate to the backend directory: cd NaturalResourceManagement\backend
Install the required Python packages: pip install -r requirements.txt
Start the FastAPI server: uvicorn app.main:app --reload
The API will be available at http://127.0.0.1:8000.


To run the frontend:

In a separate terminal, navigate to the frontend directory: cd NaturalResourceManagement\frontend
Install the Node.js dependencies: npm install
Start the Angular development server: ng serve
The web application will be available at http://localhost:4200
